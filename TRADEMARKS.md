# Trademarks and Brand Use

This project reserves the name **Lucy TTS**, the Lucy TTS app icon, logos,
product artwork, and related visual identity as project marks.

The source code license lets you use, study, modify, and share the code under
the GNU AGPLv3-or-later. It does not grant trademark rights.

## Allowed Uses

You may:

- Refer to the project by name when accurately describing the original project.
- Say that your work is based on, forked from, or compatible with Lucy TTS.
- Use the project name in non-confusing places such as dependency notices,
  attribution, changelogs, or academic/commentary references.
- Build the official app locally for personal use or contribution testing.

## Uses That Need Permission

Please ask before you:

- Publish an app, service, package, website, or organization using the name
  "Lucy TTS" or a confusingly similar name.
- Use the Lucy TTS app icon, logo, or artwork as the branding for a modified
  distribution.
- Suggest that your fork, service, app store listing, support offering, or
  hosted version is official, endorsed, sponsored, or maintained by the Lucy TTS
  project.

## Forks and Modified Versions

Forks are welcome. Modified versions should use a distinct name and icon unless
they are official Lucy TTS releases. A clear phrase like "based on Lucy TTS" is
fine; branding a modified app as simply "Lucy TTS" is likely to confuse users.
